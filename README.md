# KotlinMultiPlatformPractice


Recipe app built using Kotlin multiplatform.

## Architecture

### 1-Shared Components

   Ktor (Network Client)

   SQL Delight (Caching Client)

   Kotlinx.datetime

### 2-Android Specific Components

   Jetpack Compose

   Jetpack Compose Navigation

   (Single activity, zero fragments)

   Accompanist Coil

   Hilt
